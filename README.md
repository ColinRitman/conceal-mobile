# Conceal Mobile

### Requirements

- NodeJS v10 or better
- Expo SDK 33 or better

### Installation

```bash
$ npm install -g expo-cli
$ git clone https://github.com/ConcealNetwork/conceal-mobile.git
$ cd conceal-mobile
$ npm install
```

### Running

```bash
$ expo start
```

### Building

```bash
$ expo build:android
# or
$ expo build:ios
```
